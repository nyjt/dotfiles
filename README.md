dotfiles
========

My dotfiles for bash, vim, gem, rspec, git, rubocop, etc.

### Installing

```bash
cd
git clone https://github.com/nyjt/dotfiles.git
cd dotfiles
./install.sh
```

## On Ubuntu

```bash
./ubuntu-laptop-install.sh
```


### Updating

```
cd ~/dotfiles
git pull origin master
./install.sh
```
